# RISC Processor Design

## Overview
This project implements a simple RISC processor with a custom 16-bit instruction set and a 4-stage execution pipeline (Fetch, Decode, Execute, Write-back).

## Features
- 8 general-purpose registers
- ALU operations (ADD, SUB)
- Shift operations (SHL, SHR)
- Memory instructions with shifted offsets
- Separate instruction and data memory
